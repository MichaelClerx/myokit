************
Introduction
************

These pages contain the documentation for Myokit. This consists of

- A :ref:`guide to using Myokit <guide>`. This section explains the basics of
  using Myokit.
- The :ref:`syntax specification <syntax>` for ``mmt`` documents.
- The documentation for the :ref:`core API <api>`. This part of the site
  documents the classes, methods and functions that make up the heart of the
  Myokit framework.
- The documentation for the :ref:`simulations <api/simulations>` provided by
  Myokit.
- The documentation for the :ref:`library <api/lib>`. This is the reference
  for all tools provided with Myokit.
- A :ref:`list of formats <api/formats>` that Myokit can import and/or export
  or otherwise work with.
- Developer documentation on the :ref:`GUI classes <api/gui>`.
- An alphabetical :ref:`index <api/index>` to the Myokit api.
- A set of :ref:`command line tools <cmd/index>`.


For more information write to michael@myokit.org
